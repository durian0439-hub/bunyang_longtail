# 경매 롱테일 키워드 리서치 요약

## 수집 기준
- 목적: `bunyang_longtail` 안에서 경매(`auction`) 도메인을 분양/청약과 같은 방식으로 자동 replenish할 수 있도록 검색 의도 기반 키워드 풀을 확장한다.
- 외부 확인: 법원경매정보(`courtauction.go.kr`), 온비드(`onbid.co.kr`)를 공식 출처 축으로 확인했다.
- 키워드 분석 사이트 확인: 블랙키위, 키워드마스터(whereispost), 아이템스카우트의 키워드 분석 서비스 접근성을 확인했다. 공개 페이지는 확인됐지만 실검색 데이터는 동적/보안 절차가 있어 자동 수집 원천으로 직접 의존하지 않았다.
- 자동완성 수집: Google Suggest, Naver autocomplete, Bing OSJSON, Daum suggest를 209개 seed로 조회했다.
- 정제 기준: 주식 `공매도`처럼 부동산 경매 의도와 다른 후보는 최종 카탈로그에서 제외한다.

## 수집 규모
- seed: 209개

## 운영 반영 원칙
- broad keyword가 같다는 이유만으로 중복 차단하지 않고, domain + semantic topic/title/slug 기준으로 중복을 막는다.
- 경매 글은 법률·세금·대출·수익을 단정하지 않고 매각물건명세서, 등기부등본, 현황조사서, 감정평가서, 전입세대열람 등 확인 순서를 안내한다.
- 지역 키워드는 후보 풀 확장을 위해 포함하되, 발행 선택은 family/priority/seo_score 기반으로 분산한다.
